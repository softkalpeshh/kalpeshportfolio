## 🛠️ Built With
- HTML5 & CSS3
- Vanilla JavaScript
- [Typed.js](https://github.com/mattboldt/typed.js/) — animated text
- Google Fonts (Space Grotesk, Syne)

## ✨ Features
- Responsive design with mobile hamburger menu
- Animated typing effect (Web Developer → Data Scientist → ...)
- Scroll reveal animations
- Work experience timeline
- Skills & Technologies grid
- Contact section with email & LinkedIn

## 📁 Sections
- **Hero** — Introduction with animated role titles
- **About** — Background and personal info
- **Projects** — Featured work
- **Experience** — HTML Dev → Node.js → J.P. Morgan → YouTube → Ralph Lauren
- **Skills** — Tech stack (React, Node.js, Python, TensorFlow, etc.)
- **Contact** — Email and social links
